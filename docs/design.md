## 6. 애니메이션

| 요소 | 효과 | 트리거 | duration |
|------|------|--------|----------|
| 섹션 진입 | fade-in + slide-up (12px) | Intersection Observer | 600ms |
| 네이버 리뷰 카드 | fade-in + slide-up | Intersection Observer | 600ms (stagger 100ms) |
| 버튼 hover | background 전환 | hover | 150ms ease |
| 카드 hover | box-shadow 전환 | hover | 200ms ease |
| 모바일 메뉴 | slide-down | 클릭 | 250ms ease |
